# Sistema de Credenciamento de Profissionais - Unimed

Sistema corporativo para cadastro e análise de profissionais da saúde.

## Estrutura do Projeto
- `backend/`: API REST desenvolvida em Django.
- `frontend/`: Aplicação Web desenvolvida em React + Vite.

## Como Iniciar
Consulte o arquivo [INFRA_GUIDE.md](./INFRA_GUIDE.md) para instruções detalhadas de setup com Docker.

## Tecnologias
- **Backend**: Python 3.12, Django 5, DRF.
- **Frontend**: React, TypeScript, Vite.
- **Banco de Dados**: PostgreSQL.
